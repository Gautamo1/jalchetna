# Complete List of India States/UTs with Districts for WRIS Ground Water Level API

This is a comprehensive reference document listing all states and union territories of India with their respective districts. Use any state name with its corresponding district name in the India WRIS Ground Water Level API query.

**API Query Format:**
```
https://indiawris.gov.in/Dataset/Ground Water Level?
stateName=STATE_NAME&
districtName=DISTRICT_NAME&
agencyName=CGWB&
startdate=2023-11-01&
enddate=2024-10-31&
download=false&
page=0&
size=1
```

---

## Usage Notes

1. **Exact Spelling**: Use the district names exactly as listed for the API query to work properly
2. **agencyName**: Always use "CGWB" for Central Ground Water Board data
3. **Date Format**: Use YYYY-MM-DD format for start and end dates
4. **No Monitoring in 4 Regions**: CGWB does not have monitoring wells in:
   - Sikkim
   - Mizoram
   - Manipur
   - Lakshadweep

5. **Data Collection Seasons**:
   - Pre-monsoon: March, April, May
   - Monsoon: August
   - Post-monsoon: November
   - Winter: January

6. **National Hydrograph Network Stations (NHNS)**: CGWB operates approximately 25,000 monitoring stations across India
